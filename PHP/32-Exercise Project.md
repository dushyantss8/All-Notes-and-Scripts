# 2. Practice Project: CSV-Based Expense Tracker Component

The section concludes with a hands-on mini-project designed to reinforce the concepts learned so far.

## 2.1 Project Objective

Build a small component of a budgeting or expense-tracking application using only procedural PHP (object-oriented concepts will be introduced later).

## 2.2 Scope of the Project

You will implement a simple CSV file parser that performs the following tasks:

1. Accept one or multiple CSV files as input.
2. Parse each file and extract financial transactions.
3. Convert the parsed data into structured arrays in memory.
4. Display the transactions in a simple HTML table (no advanced UI needed).
5. Prepare for later enhancement where transactions will be stored in a database.

## 2.3 Skills Practiced

This mini-project reinforces:

* File system operations (opening, reading, parsing CSV files).
* Working with arrays, loops, and conditionals.
* Generating HTML output using PHP.
* Applying data-handling techniques (cleaning, structuring, categorizing).
* Using the foundational PHP features covered so far.
