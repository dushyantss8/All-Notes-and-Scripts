# Project Packages

| Package              | Description                                                    |
| -------------------- | -------------------------------------------------------------- |
| `bcryptjs`           | Hashes and verifies passwords.                                 |
| `compression`        | Compresses HTTP responses using Gzip.                          |
| `cookie-parser`      | Parses cookies from incoming requests.                         |
| `cors`               | Enables Cross-Origin Resource Sharing (CORS).                  |
| `dotenv`             | Loads environment variables from `.env` files.                 |
| `express`            | Web framework for building APIs and web applications.          |
| `express-rate-limit` | Limits repeated requests to prevent abuse.                     |
| `express-slow-down`  | Slows excessive requests instead of blocking them immediately. |
| `helmet`             | Adds security-related HTTP headers.                            |
| `jsonwebtoken`       | Creates and verifies JWT authentication tokens.                |
| `morgan`             | Logs incoming HTTP requests.                                   |
| `multer`             | Handles multipart/form-data and file uploads.                  |
| `mysql2`             | MySQL database driver for Node.js.                             |
| `nodemailer`         | Sends emails from the application.                             |
| `sequelize`          | Promise-based ORM for SQL databases.                           |
| `sharp`              | Processes and optimizes images.                                |
| `swagger-jsdoc`      | Generates Swagger/OpenAPI documentation from code comments.    |
| `swagger-ui-express` | Serves Swagger UI for API documentation.                       |
| `uuid`               | Generates unique identifiers (UUIDs).                          |
| `winston`            | Logging library for application logs.                          |
| `xss`                | Sanitizes input to prevent XSS attacks.                        |
| `zod`                | Schema validation and type-safe data validation library.       |
