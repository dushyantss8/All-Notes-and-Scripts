# Javascript Cheat Sheet

- Prefer `const`; use `let` for reassignment.
- `===` avoids coercion surprises; `Object.is` handles `NaN`.
- `Map` for keyed data, `Set` for membership.
- Event loop: synchronous work, then microtasks, then next timer/I/O turn.
- Spread is shallow; clone deliberately.

**Explain:** closures, `this`, prototype lookup, Promise errors, `AbortController`.
