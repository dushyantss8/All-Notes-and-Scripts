# Asynchronous JavaScript Exercises

1. Create an error-first timer callback.
2. Refactor a nested callback flow into named functions.
3. Promisify a callback API.
4. Make a callback asynchronous consistently.
5. Chain fetch, validation, and persistence.
6. Recover from a rejected Promise.
7. Compare all four Promise combinators.
8. Add a timeout race and cleanup.
9. Parallelize two independent awaits.
10. Add `AbortSignal` support.
11. Predict event-loop output.
12. Demonstrate microtask starvation safely.
13. Read a file as a stream.
14. Transform each stream chunk.
15. Handle pipeline failure.
16. Implement a backpressure-aware writable.
17. Encode/decode UTF-8 Buffer data.
18. Parse fixed-width binary fields.
19. Explain Buffer view sharing.
20. Build a retry with exponential backoff.
21. Limit concurrent Promise work.
22. Implement an async iterator consumer.
