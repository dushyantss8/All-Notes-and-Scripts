# ES6 Exercises

Solve without mutating inputs unless the prompt permits it. Add tests for empty input, `null`, and unusual values.

1. Replace `var` in a loop and explain the closure result.
2. Make a configuration object shallowly immutable.
3. Convert a callback while preserving its receiver.
4. Build a tag that escapes HTML interpolations.
5. Extract nested API data with defaults.
6. Rename destructured values without shadowing.
7. Clone an array, then demonstrate why it is shallow.
8. Write `sum(...numbers)` with rest parameters.
9. Merge defaults with user options safely.
10. Convert a CommonJS module to ESM.
11. Create a default export and a named export.
12. Build a class with a private counter.
13. Extend it and call `super`.
14. Wrap a callback API in a Promise.
15. Use `Promise.all` while preserving result order.
16. Handle a rejected `await` with contextual error data.
17. Use a Symbol to hide metadata from `Object.keys`.
18. Count words with a `Map`.
19. Deduplicate objects by ID using `Set`/`Map`.
20. Read `response?.data?.items ?? []`.
21. Lazy-load a formatter with `import()`.
22. Implement a finite generator.
23. Create a validation Proxy.
24. Add two BigInts and reject mixed numeric types.
