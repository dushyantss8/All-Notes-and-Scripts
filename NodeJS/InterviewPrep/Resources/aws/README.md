# AWS: EC2, S3, IAM

EC2 runs compute; S3 stores objects; IAM grants narrowly scoped identities. Prefer roles over static keys, private networking where appropriate, server-side encryption, lifecycle policies, CloudWatch telemetry, and presigned URLs for direct S3 upload.

**Interview angles:** security groups, autoscaling, load balancers, S3 consistency/cost, IAM least privilege, and incident containment.
